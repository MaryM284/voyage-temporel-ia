# IA Temporal — Voyage Temporel

Démonstration technique d'une interface de réservation de voyages temporels assistée par Intelligence Artificielle, développée dans le cadre du module d'intégration web.

## 🌍 Liens du Projet

* **Webapp Déployée :** [https://marym284.github.io/voyage-temporel-ia/](https://marym284.github.io/voyage-temporel-ia/)
* **Dépôt GitHub :** [https://github.com/MaryM284/voyage-temporel-ia](https://github.com/MaryM284/voyage-temporel-ia)

## 🛠️ Spécifications Techniques

* **HTML5 :** Structure sémantique complète de la Landing Page (Hero, Époques phares, Module IA, Chatbot Chronos, Formulaire de réservation et Footer).
* **CSS3 :** Architecture Responsive et stylisation avancée (Grid Layout, Flexbox, dégradés cyberpunks, intégration de typographies et gestion des états actifs).
* **Déploiement :** Intégration et hébergement automatisés via la branche principale sur **GitHub Pages**.

## 📌 Note d'intention
L'intégration respecte scrupuleusement la charte graphique et la structure globale de la maquette Figma originale. Par souci de livraison continue, une version structurelle stable a été déployée en priorité. Des optimisations d'interactivité dynamique (JavaScript) et des micro-animations CSS avancées seront poussées lors des prochaines itérations sur le dépôt afin de parfaire l'expérience visuelle.
